# shiftr
 
A customized version of shift it window mangagment written in Lua with Hammerspoon for Mac OSX.
command+option key combos for single hand use.
[     50% left side <br>
]     50% right side <br>
 <br>
j     50% Top side <br>
k     50% bottom side
L     30% left side
;     50% middle 
'     30% right side

<     70% left side
>     70% right side

n     toggle fullscreen app
m     maximize window

splt screen + basic multi screen 'scenes'
